## Features

- **Track TV shows** by season and episode (S3 E7)
- **Track movies** by timestamp (01:12:33)
- **Sidebar filters** with live badge counts &mdash; Watching, On Hold, Finished, Rewatching, by type
- **Instant search** &mdash; hit `Cmd+F` and start typing, results filter in real-time
- **One-click progress** &mdash; next episode, next season, mark finished, start rewatch
- **Inspector panel** &mdash; detail view with quick actions, notes, and metadata
- **Keyboard shortcuts** for every action &mdash; your hands never leave the keyboard
- **Auto-updates** via Sparkle &mdash; always on the latest version
- **Customizable appearance** &mdash; row density (compact/comfortable) and accent colors
- **Launch at login** &mdash; always ready when you are
- **Local-first** &mdash; no accounts, no cloud, no tracking. All data stays on your Mac

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | Add new item |
| `Cmd+F` | Search |
| `Cmd+Return` | Next episode |
| `Cmd+Delete` | Mark finished |
| `Cmd+E` | Edit item |
| `Cmd+Opt+I` | Toggle inspector |
| `Cmd+,` | Settings |

## Installation

### Download

Grab the latest `.dmg` from [GitHub Releases](https://github.com/owldesign/ShowTracker/releases/latest), open it, and drag ShowTracker to your Applications folder.

### Build from Source

```bash
# Clone the repo
git clone https://github.com/owldesign/ShowTracker.git
cd ShowTracker

# Install xcodegen (if you don't have it)
brew install xcodegen

# Generate the Xcode project
xcodegen generate

# Open in Xcode and build (Cmd+R)
open ShowTracker.xcodeproj
```

## Project Structure

```
ShowTracker/
  ShowTrackerApp.swift              # App entry point, model container, scenes
  Info.plist                        # Generated by xcodegen (Sparkle keys, etc.)
  Models/
    WatchItem.swift                 # @Model with indexes and mutations
    Enums.swift                     # WatchType, WatchStatus, RowDensity, AppAccentColor
  Views/
    ContentView.swift               # NavigationSplitView shell, state owner
    SidebarView.swift               # Filter sidebar with badge counts
    WatchListContainer.swift        # Searchable list + toolbar + inspector
    FilteredWatchList.swift         # Dynamic @Query with #Predicate
    WatchItemRow.swift              # List row with density support
    WatchItemDetailView.swift       # Inspector detail + quick actions
    AddEditSheet.swift              # Modal add/edit form
    Settings/
      SettingsView.swift            # 3-tab settings (General, Appearance, About)
      GeneralSettingsView.swift     # Launch at login, update settings
      AppearanceSettingsView.swift  # Row density, accent color
      AboutView.swift               # Version info, links
  Updater/
    UpdaterController.swift         # Sparkle wrapper (@Observable)
  Commands/
    AppCommands.swift               # Menu commands + FocusedValues
  Preview Content/
    PreviewSampleData.swift         # In-memory sample data for previews
project.yml                         # xcodegen project spec
scripts/
  release.sh                        # Build, DMG, sign, appcast generation
  generate_icon.py                  # App icon generator
site/                               # Landing page (GitHub Pages)
```
